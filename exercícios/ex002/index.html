Isso é um texto de exemplo com quebra de linha.
<br> significa: break roll .

E esse espaço entre um paragrafo e outro é automatico, não tem nenhuma tag br entre os dois.

Esse é o segundo paragrafo e mesmo no código HTML eu quebrando linha com enter não vai adiantar nada na hora de aparecer no navegador.

Para adicionar símbolos como < e > é necesario usar "& l t ; e & g t ; ".
→ ↓ ← ↑
Setas feitas com & + r/d/l/u + arr;
&: Inicio dos símbolos.
r/d/l/u: right/ down/ left/ up.
arr: Abreviação de setas.

Vamos adicionar alguns simbolos: δ Δ € ¥ £ ¢ © ® ™ Símbolos são feitos com & + nome abreviado + ;

Vamos adicionar alguns emojis: 🚀 🌌 🛸
Emojis são feitos com &#x + código do emoji + ;
